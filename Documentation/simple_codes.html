1. Columns - top

Columns are the most fundamental element of your layouts. They are fully width-adjustable so you can use them on full-width pages aswell as on the sidebar layout content and even on the sidebar. You may also place them inside tabs, toggles and box contents. The list is just endless.

They are also very easy to use. Here is a basic example:

<div class="one_fourth">
	<!-- One fourth column content -->  
</div>

<div class="three_fourth last"> 
	<!-- Three fourth column content -->  
</div>

<div class="clear"></div>

NOTE 1: Never forget to use an additional class "last" on the last column in the row. If you do, it will break whole page layout.

NOTE 2: Never forget to use a div ".clear" after a row of columns. If you do, it will break the whole page layout. You may also use dividers - read more about them Here.

Here is a full list of columns you may use:

  1. one_half
  2. one_third
  3. two_third
  4. one_fourth
  5. three_fourth
  6. one_fifth
  7. two_fifth
  8. three_fifth

For a better look, I included a file columns.html (located in /www/ folder) where you can see all columns in action.

2. Typography - top

You may check all the typography simple codes in action in a "typography.html" file located in your /www/ folder.

2.1. Headings - top

For your use, I prepared six different headings. Here is a basic example of an usage:

<h1>Heading </h1>
<p>Some text content</p>

Here is a full list of headings: h1, h2, h3, h4, h5, h6.

2.2. Pull quotes - top

Place them inside paragraphs:

<p>
        
..some text content here..

<span class="pullright">Right pull quote text content</span>

..some text content here..

<span class="pullleft">Left pull quote text content</span>

..some text content here..

</p>

2.3. Images styling - top

Place them inside or outside paragraphs:

<p>
        
..some text content here..

<img src="img/your_image.jpg" class="imageright" alt>

..some text content here..

<img src="img/your_image.jpg" class="imageleft" alt>

..some text content here..

</p>

2.4. Centered images - top

Place them outside paragraphs. This is a standard centered image without a border:

<img src="img/your_image.jpg" class="center" alt>

And here is a code for a centered image with a border

<img src="img/your_image.jpg" class="center border" alt>

NOTE: Since default images without a class don't have a border, you may use the class ".border" on anywhere you want.

2.5. Images with caption - top

Place them inside or outside paragraphs:

<p>
        
..some text content here..

<div class="blockright">
    <img src="img/your_image.jpg" alt>
    <p class="caption">This is a caption</p>
</div>

..some text content here..

<div class="blockleft">
    <img src="img/your_image.jpg" alt>
    <p class="caption">This is a caption</p>
</div>

..some text content here..

</p>

2.6. Text highlighting - top

Here is an usage example:

<p>
        
<span class="highlight-light">This is a light highlighted text.</span> Here is some not highlighted text. Here is some not highlighted text. Here is some not highlighted text here text  here. <span class="highlight-dark">This is a dark highlighted text.</span> Here is some not highlighted text. Here is some not highlighted text. 2.7. Drop caps - top

Place them inside or outside (for example if it's a number) a paragraph:

<p>
	<span class="dropcap1">Q</span>uisque aliquam donec faucibus. Suspendisse quam sem, consequat at, commodo vitae, feugiat in, nunc. 
</p>
<p>
	<span class="dropcap2">L</span>uisque aliquam donec faucibus. Suspendisse quam sem, consequat at, commodo vitae, feugiat in, nunc. 
</p>

2.8. Block quotes - top

A block quote element automatially adjusts it's width to the column. I suggest you to place them inside a paragraph.

A standard one:

<blockquote>
    <p>
    This is a blockquote text content.
    </p>
</blockquote>

A block quote with a quote author:

<blockquote>
    <p>
    This is a blockquote text content.
    </p>
    <span>Quote author</span>
</blockquote>

3. Elements - top

You may check all elements simple codes in action in a "elements.html" file located in your /www/ folder.

3.1. Icon boxes - top

Icon boxes are closely related with columns. They MUST be placed in one of these three: One Third, One Fourth or One Fifth. Here is a basic structure of a One Fourth icon box:

<div class="one_fourth"> 
    <div class="icon-image">
        <img src="img/icons/eye.png" alt>
    </div>
    <div class="icon-text">
        <h5>Icon box Title</h5>
        <p>Icon box description.</p>
    </div>
</div>

To create a One Fifth icon box, simply replace the "one_fourth" class with "one_fifth".

NOTE:As with the columns, never forget to use additional class "last" on the latest icon box in a row. You may not forget about a divider after every row of icon boxes or it will break the whole page layout. Please refer to Columns to learn more.

Template comes with 36 icons. Want more? Get over 400 here!

3.3. Big buttons - top

The only thing that differs from the standard size buttons is the "button" class being replaced by "big_button". Example:

<a href="http://somepage.com" class="big_button red">This is a big red button</a>

3.4. Tabs - top

Tabs automatically adjusts their width to a column. They have a bit more complicated structure. Here is the code:

<div class="tabs">

    <ul class="tabnav">
        <li><a href="#tab_1">Tab 1</a></li>
        <li><a href="#tab_2">Tab 2</a></li>
        <li><a href="#tab_3">Tab 3</a></li>
    </ul>
    
    <div id="tab_1" class="tabdiv">
        <p>
        Tab 1 content.
        </p> 
    </div>
    
    <div id="tab_2" class="tabdiv">
        <p>Tab 2 content.</p>
    </div>
    
    <div id="tab_3" class="tabdiv">
        <p>
        Tab 3 content. 
        </p>
    </div>
    
</div>

The most important thing is to give a tab navigation element the same href as an id of a tabdiv element where the tab content is.

3.5. Toggles - top

Toggles automatically adjusts their width to a column aswell. They are a bit less complex then tabs:

<ul class="toggle-view">
        
    <li>
        <h6 class="toggle">Do toggles automatically adjust their width?</h6>
        <p>Yes, this element automatically adjusts its width to the column!</p>
    </li>
    
    <li>
        <h6 class="toggle">Toggle 2</h6>                            
        <p>Toggle 2 content</p>
        
    </li>
    
    <li>
        <h6 class="toggle">Toggle 3</h6>                            
        <p>Toggle 3 content.</p>
    </li>
    
</ul>

3.6. Lists - top

There are two different list types: an unordered list (<ul></ul>) and an ordered list (<ol></ol>).

Here is an usage example of the first type, unordered list with a checklist icon:

<ul class="list check">
    <li>List element 1</li>
    <li>List element 2</li>
    <li>List element 3</li>
    <li>List element 4</li>
    <li>List element 5</li>
</ul>

There are two other styles of the unordered list: a circle list (replace the 'check' class with 'circle') and a square list (replace it with 'square').

Here is an usage example of the second list type, ordered list:

<ol>
    <li>List element 1</li>
    <li>List element 2</li>
    <li>List element 3</li>
    <li>List element 4</li>
    <li>List element 5</li>
</ol>

3.7. Content boxes - top

Content box automatically adjusts it's width to a column. Here is an usage example:

<div class="box box-grey">
    This is a grey content box.
</div>

To make your own color version, open up your custom.css file (located in /www/css/ folder) and add a rule there. For example, let's say that you want to make a red content box. Name it "box-red" and place this code in your custom.css file:

.box-red { 
  border-color:red; 
  color:red; 
}

To use a newly created box style, change the "box-grey" class I showed on the example above with your new class "box-red".

3.8. Dividers - top

Dividers automatically adjust their width to a column (which matters if you use a line or a top divider). Dividers are used to divide content areas within a web page.

A simple divider, just 'space':

<div class="divider"></div>

A line divider:

<div class="divider line"></div>

A line divider with the 'Top' linke:

<div class="divider top"><a href="#">Top</a></div>
