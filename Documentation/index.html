1. HTML Structure - top

This theme is a one column layout with three main sections: Header (#header), Main Content (#main) and Footer (#footer). All of the information within the main content area is nested within a div with a class of ".inner" that is held by a div with an id of "#main". The same thing goes to footer. The information within the footer area is nested with a class of ".inner" that is held by a div with an id of "#footer".

Few basic information:

  1. Put all your text content and images (except for centered ones) inside the paragraph tags: 
    <p> Your text here </p>
    If you want to make an another paragraph, don't use the double break tag (<br>). Instead of that, close the previous paragraph and start a new one. 
    <p> First paragraph here </p>
<p> Second paragraph here </p>
  2. Don't place headings (h1, h2, h3, etc..) inside paragraphs. Place them outside: 
    <h5>Your heading here</h5>
<p> Paragraph here </p>
  3. To make an image adjust it's width to a column use this code: 
    <img src="some_image.jpg" class="full-width" alt="An alternate text for an image">

1.1. Customizing #header - top

Header consists of 2 elements: Logo (.logo) and Navigation (.nav).

1.1.1. Logo - top

To change a logo simply find a div with a class of .logo:

<div class="logo">
	<a href="#"><img src="img/logo.png" alt="Purity"></a>
</div>

All you have to do here is to change the src path to your own image.
NOTE! If your logo image height differs from the '.contact' div, you will want to center it vertically. To do that, simply open the main stylesheet file (style.css) and look for this line: 

#header .logo { float:left; padding:30px 0 0 0; }
The parameter you need to edit here is 'padding'. The .contact div is 90px height, so if your logo image height is, for example 40px, change the first padding value (30px) to 25px.

1.1.2. Navigation - top

The navigation holder (ul - unordered list) is floated to the left side and is contained within the div .inner that is held by the div .nav (.nav > .inner > ul). Here is the general structure:

<ul> 
	<li class="current"><a href="index.html">Home</a></li>
	<li><a href="about.html">About</a>
		<ul>
			<li><a href="sub1.html">Submenu 1</a></li>
			<li><a href="sub2.html">Submenu 2</a></li>
			<li><a href="sub3.html">Submenu 3</a></li>
		</ul>
	</li> 
	<li><a href="blog.html">Blog</a></li>
	<li><a href="portfolio.html">Portfolio</a></li> 
	<li><a href="contact.html">Contact</a></li> 
</ul>

Use class "current" on the current page menu item. On the above example menu item "Home" will be 'exposed'.

2. CSS Files and Structure - top

I'm using several CSS files in this theme. The first one, "style.css", is a main template stylesheet. It will make it much easier for you to edit the layout later. The second stylesheet is "custom.css" which is a place for your custom css styling. They are both located in the /css/ folder.
There are also two stylesheets for scripts located in /css/scripts/: flexslider.css and prettyPhoto.css.

NOTE: If you aren't familiar with CSS, please don't edit the "style.css" file since it may affect the whole layout. Instead of that, I suggest you to copy a selected style rule from "style.css" and put it inside the "custom.css" file. It will simply get overwritten.

2.1. General information - top

Here are some general informations about Purity styling: (As mentioned above, use the "custom.css" file for a custom styling).

  1. To change the general font rules, use this code: 
    body {
    font-size:12px;
    color:#666;
    font-family: Arial, Helvetica, sans-serif;
}

    You don't have to change all these settings of course. You may change only the font-size rule, without changing the color and font-family.

  2. To change the headings font rules, use this code: 
    h2 {
    font-size:28px;
    line-height:28px;
    color:#666;
    font-family:'LeagueGothicRegular', Helvetica, Arial;
    font-style:italic;
    margin-bottom:16px;
}

    You can also change rules for few elements at once by simply replacing, for example, 'h2' with 'h1, h2, h3, h4'. It will affect all those 4 headings.

  3. To change the homepage tagline font rules, use this code: 
    .home_tagline h1 {
    font-size:32px;
    line-height:40px;
    color:#888;
}

2.2. Using additional color stylesheets - top

The template comes also with 10 additional stylesheets:

  1. Nine color schemes that are located in /css/schemes/
  2. Dark skin stylesheet (dark.css) located in /css/skins/

You can use these additional schemes in two ways:

  1. Attach the stylesheet to the <head></head> section of your website. For example, if you would like to use a layout with a dark skin and a red color scheme, add this code: 
    <link rel="stylesheet" href="css/skins/dark.css" type="text/css" media="screen">
<link rel="stylesheet" href="css/schemes/red.css" type="text/css" media="screen">
  2. Second way is to copy the selected stylesheet content and paste it inside the custom.css file. For example, to get the same effect as above, copy the red.css content, paste it to the custom.css file and to the same thing for dark.css file. This way you can save some space in your <head></head> section.

To change a background image or color, add this code to your custom.css file:

body {
  background:#222 url(../img/backgrounds/image1.jpg) no-repeat top center fixed;
}

Where '#222' is a default background color that appears before the background image is loaded. You may also not want to use any background image and have it as a plain color. To do that, simply remove the rules after the color part (#222), just like that:

body {
  background:#222;
}

If you want to know more about CSS background rules, Visit this site.

2.3. Changing the heading font - top

The heading font is the BOLD one and is implemented using the @font-face. It is named League Gothic and you can find it Here.

Here is a guide of how to change the Heading Font:

  1. Go to Font Squirrel and choose a font. On the font page, navigate to the @font-face tab and download the @font-face Kit to your hard disk.
  2. Unrar the archive and open the stylesheet.css file. Copy the whole @font-face rule of the font weight that you want (bold, normal, italic, there will be plenty to choose from) and paste it (replace with the old one) to the Purity's main stylesheet (/css/style.css - 4.1.1. Heading Font section). As you are done with that, change the src paths in the rule. Since font files are stored in the /font/ folder, you need to add it at the beginning.

    For example you should change this: 
    url('League_Gothic-webfont.eot');

    To this: 

    url('font/League_Gothic-webfont.eot');
  3. Just under the @font-face rule in the main Purity stylesheet (/css/style.css) there is a rule that defines the heading font-family. It looks like this: 
    .nav,h1,h2,h3,h4,.home_tagline h1,.tagline h1,.over span,.dropcap1,.dropcap2,.bold,.flex-caption { 
	font-family:'LeagueGothicRegular', Helvetica, Arial; 
	text-transform:uppercase; 
	font-weight:normal; 	
}

    All you need to do here is change the 'LeagueGothicRegular' name to the name of your font that is defined in the @font-face rule (font-family part).

    As you may see, there are also two font attributes: text-transform (in this case it transforms all letters to the uppercase) and font-weight (normal weight here). Change them as you wish.

  4. Go to the folder you just downloaded again and copy the font files that are mentioned in your new @font-face rule and paste them to the /css/font/ folder of your page.

NOTE: Since League Gothic is a pretty 'small' font I had to use pretty big font sizes to make it look good enough. To change the font size, look for the rules of the tags mentioned in the 'heading font definer' line (the one where you just set the font-family name).

	    <li><img src="img/pictures/6w.jpg" alt="" /></li>
	</ul>
</div>
As you can see, the slider structure is very simple.

  1. To add a slide, use this code: <li><img src="slide_image_source.jpg" alt="Optional title" /></li>
  2. To add a slide with a caption, use this code: <li><img src="slide_image_source.jpg" alt="Optional title" title="Optional caption text" /><li>

If you would like to change a transition effect, pause time and other slider settings, open "custom.js" file (located in /js/ folder) and find FlexSlider lines:

jQuery('#slider').flexslider({
	slideshowSpeed: 4000,
	animationDuration: 700,			
	directionNav: true,
	controlNav: true,
	animation: 'fade'
});

The time unit here is a milisecond (1 second = 1000ms), so '4000' stands for 4000 miliseconds which is equal to 4 seconds. If you want to change more options of the FlexSlider, Check this website.

5. Installing Cufon (Firefox 2.0+ compatibility) - top

At first, I'd like to present you the cons and pros of using this method:

Pros:
  1. Firefox 2.0+ compatibility
  2. Better rendering quality when it comes to more 'shapeful' fonts
Cons:
  1. Worse performance comparing to @font-face
  2. Text cannot be selected
  3. No 'smooth color transition on hover on links in modern browsers

I listed only arguments that may really interest you. You need to answer yourself if it is worth just for those 0,4% of all Firefox users (Firefox Stats)?

Installation guide:

  1. Add these three javascript to the bottom section (right where other javascripts are, just don't place them above the jquery one) on all your pages: 
    <script type="text/javascript" src="js/cufon/cufon-yui.js"></script>     
<script type="text/javascript" src="js/cufon/cufon-config.js"></script> 
<script type="text/javascript" src="js/cufon/League_Gothic.font.js"></script>

    As you may see, the last file is responsible for the font itself. Learn more how to get more fonts with Cufon Here and just replace it with your own.

  2. You also need to edit some CSS to fix the Navigation hover color. Open the main stylesheet (/css/style.css) and completely remove this line from the Navigation section of the style.css: 
    .nav ul li:hover > a { color:#d16f4e; }
  3. Since Cufon doesn't support the 'smooth transition' you may want to remove it completely. Find this line: 
    a { color:#d16f4e; text-decoration:none; transition: color 0.1s linear; -moz-transition: color 0.1s linear; -webkit-transition: color 0.1s linear; -o-transition: color 0.1s linear; }

    And change it to this:

    a { color:#d16f4e; text-decoration:none; }

    It also appears on the .comments-nr .bold tag so find this line:

    .date .bold,.comments-nr .bold { font-size:28px; line-height:28px; display:block; color:#444; transition: color 0.1s linear; -moz-transition: color 0.1s linear; -webkit-transition: color 0.1s linear; -o-transition: color 0.1s linear; }

    And change it to this

    .date .bold,.comments-nr .bold { font-size:28px; line-height:28px; display:block; color:#444; }
  4. You will also need to edit the /js/custom.js file. Find these lines: 
    $list.quicksand($sorted_data, $preferences, function () {
	over();
	hover_opacity();
	prettyPhoto();
});

    This script reloads the over and prettyPhoto function after the portfolio is sorted. Cufon will also need to be reloaded and to do that, simply add another callback "cufon();" above those three.

    NOTE: Don't forget to remove it if you decide to not use Cufon on your site. If you do, the portfolio sorting script will not work properly.

  5. Another con of using cufon is a hover color of the 'Cufon'ed' text. You cannot change it in CSS, you have to do it through the /js/cufon/cufon-config.js file. It is well commented so you shouldn't have any problems configuring it.

Now you can enjoy the Cufon on your website.
